Metrevs

Is a automatic way to get key performance indicators which are used to see and track the developers activity, we report Github crucial statistics, these are used for the enginering team or developers to check the performance of their teams or their own

Features

Fast reports


Generate reports for your repositories in the fastest way and get detailed statistics in seconds.

Specific reports


See your repositories statistics selecting many repositories as you need, creating many teams as you want, and selecting a specific date range.

Unlimited reports and charts


Generate many reports/charts as you need or want! Our platform is free and have not limits for you!

This project was created due to the request of the Torre company, which posed a problem when it came to monitoring the developers performance; initially an integration of two APIs was expected, which were used to know the performance and time taken in every github issue. The scope of our project changed drastically wen they informed us that we had to implement a solution that does not involve the two APIs, instead, create a solution that allows obtaining detailed performance statistics about the developers according to the information collected from github in the pulls requests.

We decided to face the challenge, despite the big changes as the days go by, we were getting closer to the final product, we are proud to say that this project will continue to be developed, and you can expect news from us.

Something we learned in this journey and we are grateful, is to manage projects and create efficient work plans, which made us to understand each part of the project in depth and give us greater productivity when developing.
